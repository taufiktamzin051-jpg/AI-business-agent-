# 🤖 AI Business Agent

AI business agent untuk otomatisasi produk digital menggunakan Gemini API.

## Fitur
- 🧠 Self-assessment & adaptive planning
- ⚡ Goal-oriented execution
- 🔄 Continuous learning from experience
- 📊 Generate ide produk digital otomatis

## Cara Setup
1. Tambahkan `GEMINI_API_KEY` ke GitHub Secrets
2. Push ke branch main
3. Workflow akan jalan otomatis

## Hasil
Ide produk digital tersimpan sebagai `produk_terbaru.md` dan bisa di-download dari Artifacts.
